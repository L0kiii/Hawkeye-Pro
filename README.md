## 用法须知

支持识别markdown标题 用于任务名称

规则目前仅为url 需要添加规则参考`https://github.com/L0kiii/Github-Monitor-Pro`

基于服务器考虑 把任务都设为False 即为❎关闭按钮 如需立即启动 更改代码中`{.... 'enable':False}`为`True`

使用前需要添加账户密码

```python
python3 Hawkeye.py
python3 HawkeyePro.py
python3 HawkeyeLogin.py
```
